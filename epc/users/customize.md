# Customizing EPC

> This chapter is the mdBook version of [`CUSTOMIZE.md`](../../CUSTOMIZE.md) at the repo
> root. That file is the canonical user-facing reference; this chapter provides the same
> content in navigable form.

See [`CUSTOMIZE.md`](../../CUSTOMIZE.md) for the full customization guide, including:

- **Port 1: Services** — deploying EPS packages as persistent daemons
- **Port 2: Supervisor** — swapping the built-in process loop for launchd or systemd
- **Port 3: Dashboard** — adding a web UI (not shipped by default)
- **The state file** — `~/.epc/services.toml`, the source of truth for running services
