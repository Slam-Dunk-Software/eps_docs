# MCP Setup

> **The eps_mcp MCP server has been retired.** EPS knowledge is now delivered through Claude Code slash commands instead.

## Install eps_skills

```bash
epm install eps_docs
epm skills install eps_skills
```

This gives you four knowledge commands:

| Command | What it does |
|---|---|
| `/eps` | EPS overview — what it is, philosophy, getting started |
| `/eps-adr <n>` | Read an Architecture Decision Record |
| `/eps-toml` | Full eps.toml manifest reference |
| `/eps-epc` | EPC (process supervisor) docs |
